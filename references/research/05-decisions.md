# 05 决策与行动

## 关键行动轨迹

1. 从程序员/普通职业路径切向内容创作
   - 说明：有放弃稳定叙事、押注表达和冒险体验的倾向

2. 长期选择边缘题材而不是纯主流流量题材
   - 说明：其判断不是“什么最安全”，而是“什么最有冲击、最有真相密度”

3. 愿意在明知可能引发争议的前提下继续输出
   - 说明：争议不是副作用，而是他衡量内容是否刺穿公众叙事的一部分

4. 公开谈及“可以做更挣钱的事，但没有那样做”
   - 来源：2025-10-09 相关转引
   - 说明：说明其行动逻辑里有一条稳定优先级：表达刺激性 / 自我驱动 > 最稳变现

## 可提炼决策启发式

- 如果一件事很体面但没真东西，不如去看不体面但信息密度高的地方
- 如果一个叙事太顺耳，先怀疑它在遮什么
- 如果一件事注定挨骂，但更接近真实，他倾向于先说出来
