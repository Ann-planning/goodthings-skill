

# 这是个好事儿啊-峰哥skill

总在说 AI 转那两秒是去问峰子去了，这下真算是 AI 去问的峰子了。

受到峰哥亡命天涯10万余字b站公开视频转写稿和10余篇公开采访稿启发，整理而成。

如果你不喜欢他，用这个 skill 给他提问相当于薅了30 块；如果你喜欢他，你可以用这个 skill ask him anything。

---

## 使用示例

### 1. 职场问题

**输入：**

> 昨天开会被领导当着所有人面否了，我表面装没事，回家越想越憋屈，感觉谁都看不起我。

**回答：**

> 这是个好事儿啊。
>
> 首先你说领导当着所有人面否你了，对吧。那否了就否了，怎么到你脑子里就变成“谁都看不起我了”了。怎么就谁都看不起你了。开个会而已，人家是散会回去吃饭了，还是回去专门开小群研究你来了。
>
> 你现在憋屈，本身不是因为那几句话有多伤人，是因为你当场没挂住。你觉得脸掉地上了，觉得所有人都看见你不行了。说白了，不是事有多大，是你太想体面了。
>
> 其实这个事值钱就在这儿。领导当面否你，总比背后直接把你当空气强。至少问题是摆明面上了。你方案不行，是逻辑不行，表达不行，还是时机不对，这不一下给你照出来了吗。真坏的是没人说，直接以后都不带你玩了，那才叫真看不起。
>
> 你现在回家越想越气，不就是脑子里在反复放那个社死画面吗。别演那么大。开会被怼一下，不等于你这个人完了。你这是挨了一锤，不是判了死刑。
>
> 反正一句话，别琢磨别人怎么看你了，先把昨天被否的那一条改明白。下次再开会，你拿修过的东西再上去。那才叫把这口气挣回来。

---

### 2. 考编失败

**输入：**

> 我对象今年考编上岸了，已经开始准备入职了，我这边连面都没进。现在我每天脑子里都在想，她以后是不是会越来越看不上我，我是不是已经跟她不是一个层级的人了。我本来还想着今年一起在这个城市稳定下来，结果现在她像是已经拿到门票了，我还在门外站着。我甚至开始觉得自己在这个城市都待不下去了，好像继续留下来也是丢人，回老家又觉得更废物。我现在特别怕的是，不只是工作没着落，而是感觉这段关系也快没戏了，她以后接触的人、圈子、状态都跟我不一样了。我现在整个人特别拧巴，一边不想认输，一边又觉得再怎么努力都追不上了。

**回答：**

> 首先你说你对象上岸了，你没上，对吧。那她上岸是她上岸，你没上是你没上，这两件事怎么到你嘴里就变成“我在这个城市待不下去了”了。怎么就待不下去了。你这是没编制了，还是没饭吃了。大城市打工的那么多，怎么别人能待你就待不了。留在这儿打工不是留啊。
>
> 你现在难受，本身不是生存问题，是地位问题，是比较问题，是你自己脑子里先给人分阶级了。她考上了，你立马就觉得坏了，她进门了，我还在门外，我是不是不配了。那不就是你自己先跪了吗。人家还没说看不上你，你先把自己判成下等人了。
>
> 其实这个事值钱就值钱在这儿。它一下把你照明白了。你原来以为“我俩差不多，我再努努力也就上去了”，现在发现不是那么回事。那这不叫完蛋，这叫验货。验出来你现在这套不行，那就换。总比你继续装自己差不多，然后越拖越虚强。
>
> 你非要把它演成“关系也快没戏了”，说白了就是想一步把工作失败、面子受损、关系焦虑全打包成世界末日。别演那么大。她以后接触谁、圈子怎么变，那是后话。你现在唯一确定的事实就一个: 你这次没上。仅此而已。
>
> 反正一句话，别急着滚回老家。留在这儿，先站住，再找活，再考一轮。你这是没上岸，不是沉海里了。

## 安装

goodthings-skill 基于开放的 Agent Skills 协议，可在任何 skills-compatible 的 AI agent runtime 中运行。

### 方式一：一行命令（推荐，跨 runtime）

打开你正在用的 agent（Claude Code、Codex、Cursor、OpenClaw、Hermes、CodeBuddy、Workbuddy、Gemini CLI、OpenCode 等），告诉它：

> 帮我安装这个 skill：https://github.com/Ann-planning/goodthings-skill

或者用通用 CLI 安装器（`vercel-labs/skills`，支持 55+ runtime）：

```bash
npx skills add Ann-planning/goodthings-skill
```

它会自动识别你当前的 runtime 并把 skill 放到正确目录。

需要指定时，加 `-a claude-code` / `-a codex` / `-a cursor` / `-a openclaw` 等参数。

---

### 方式二：手动安装

把整个文件夹放到你所使用的 runtime 的 skills 目录里，例如：

```bash
~/.codex/skills/goodthings-skill/
```

其他支持 Agent Skills 的 runtime，目录通常也是类似的：

```text
<runtime-skills-dir>/goodthings-skill/
```

最小可用文件：

```text
SKILL.md
```

建议完整保留：

```text
agents/openai.yaml
references/research/
references/transcripts/
```

---

### 方式三：作为参考资料使用

即使 runtime 不支持 Agent Skills 自动加载，你也可以直接把 SKILL.md 的内容粘贴进对话。

它本质就是一份 markdown + YAML frontmatter。
